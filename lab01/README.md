# Тема лабораторной: порождающие паттерны

## Предметная область: RPG-игра

Проблема: существует некая игра, позляющая выбирать мир (Фентези, Киберпанк), а в каждом мире существует определенное наполнение (Драконы или Роботы). 

Решение: используя паттерн Абстрактная фабрика, мы создаем интерфейс, через который реализуются миры, и еще интерфейсы наполнения, через которые реализуются конкретные предметы мира.
